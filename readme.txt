### MagicBox Features

### SMTP
You can add any mail account information you want so that e-mails sent through your website can be forwarded.
You can easily add the SMTP mail information.
With this feature, messages from the contact forms on your site are forwarded to the e-mail you want.
With this feature, new membership, order, lost password, etc. E-mails are forwarded to your members.

### Contact Buttons
You can add contact and social media buttons to your website.
You can add contact icons such as Facebook, Instagram, phone, mail.
You can show the contact buttons at the top, right, bottom positions of your page.
You can adjust the settings of the buttons as you wish.
You can change the opening type of the dialog.
You can specify whether the communication widget is displayed on or off
You can show the communication widget on any page of your site.

### Form Builder
You can add as many contact forms as you want without the need for any code for your website.
You can add the forms you add to the page you want with Shortcode.
You can follow the messages sent to the forms from your admin panel.
If you want, incoming forms are automatically forwarded to the e-mail addresses you specify.

### Bulk Mail
You can send mass mail to your members or to the lists you want from your Admin Panel.
You can easily make mail designs with the editor.
You can send mass e-mails specific to the member types on your site.
You can add your mailing list in bulk manually.
You can review the details of the mass mails you have sent.
You can add special tags such as -name, -username, -mail to your maills.

### Lazy Load
You can decrease the page opening time by not automatically loading the content that your visitors do not see.
You can choose which pages the lazy load will be valid on.
You can set which content types the lazy load is valid for.

### Database Repair
You can periodically repair corrupted tables to improve your website performance and database health.
You can view the corrupted tables in your database by severity.
You can repair your tables individually or in bulk.

### Cache Manager
By caching data on your website, you can reduce your site's load time and improve your site's SEO and overall user experience.
You can set how caches are stored on your server.
You can set the duration of caches on your server.
You can set which states will be cached.
You can separate desktop and mobile user caches.
You can set the data that you do not want to be cached.

### Preloader
You can upload page load animations to your website.
You can select the preloader image you want from the gallery or add your own
You can specify the display time, size and effect of the image.
You can select the pages you want the Preloader to appear on.

### Popup
You can add automatic pop-up dialogs to your website.
You can edit the popup content with the editor.
You can edit the appearance of the popup.
You can add javascript tracking tags to your popup.
You can choose how many times the popup will appear and for how long.
You can select the pages you want the popup to open.

### Live Search
By improving the search form of your website, you can show your visitors fast and image results.
You can include images in search results.
You can limit the length of titles and descriptions.
You can specify the number of results to show.

### Fancybox
You can display the media on your website in aesthetically pleasing Fancyboxes.
FancyBox's image, video, pdf etc. You can choose which objects to apply to.
You can edit the appearance of the FancyBox.
You can select the pages on which the FancyBoxes will be displayed.

### Custom Css
You can easily customize the look of your site by adding custom CSS codes without changing your theme files.
You can add custom css to both the WordPress Admin Panel and your website.
You can add custom CSS to mobile, tablet and desktop views.

### Cookie Policy Options
You can add a dialog message that informs your visitors about the use of cookies on your site and asks for confirmation.
You can add a title and description to the cookie notice.
You can customize the appearance of the pop-up notification.
You You can edit the cookie confirmation button design.

### Login Logs
You can view the users who have attempted to login to your website.
You can view the date users attempted to login.
You can view whether the login attempt was successful or not.
You can see which ip and browser the users are trying from.

### Php Information
You can view detailed information about the server and software of your WordPress website.
You can view a lot of information about the current status of your php configuration
You can view which extensions are active on your server.
You can view the server and cookie information of your website.

### Error Logs
You can view the errors that have occurred on your website or server.
You can see the location and date of the error logs.
You can easily clear the records.

### Post Authentication
You can organize your post and page accesses for your WordPress website depending on user types.
You can only grant logged-in users content access to certain pages and posts.
You can add password restrictions to your content.

### Disqus Comment
You can add the Disqus comment system to your website and directly interact with your target audience to increase your traffic and audience.
You can adjust the position and size of the comment section.
You can activate the captcha feature while commenting.
You can limit the number of times users can comment.

### Comment Manager
You can determine in which sections the Wordpress Comment system will work on your web page, and you can prevent spam posts.
You can prevent commenting on the content you have selected.
You can hide old comments made on the content you selected.
You can activate the captcha feature while commenting.
You can limit the number of times users can comment.

### Backup Manager
You can easily back up your website against any security threats, corruptions and updates.
You can back up your entire website, plugins, themes, upload folder or database if you want.
You can see the list of backups you have taken and download the ones you want.

### Image Upload Optimiser
You can resize your images in your own browser and upload them to your site in an optimized way, without the need to resize them one by one.
You save time on your bulk image uploads.
You can set the maximum dimensions and quality of the image you will upload.
By reducing the size of your images before uploading them to your server, you consume minimal space in your hosting space.

### Bulk Image Resize
You can reduce server space usage and speed up your website by resizing images.
You can resize images of content, products etc.in bulk on your site.
You can reduce the quality of images in bulk.
You can optimize JPG, PNG and GIF files.

### Keyword Suggestion
While adding content such as posts, pages and products to your website, you can use relevant keyword suggestions to get more traffic to your content.
You can add word suggestions directly to your content or use them as a tag.
You can easily find popular keywords related to the topic.
You can target keywords correctly with this feature.

### Url Replacement
If you have changed your website address or want to change the links on your site, you can easily replace the old links with the new URL.
You can change the links of all content types collectively or individually.
You can change an old url on your site with a new url

### Sitemap
You can create dynamic XML sitemaps to help search engines better index your site.
You can create separate sitemaps depending on the content types of your website.
You can include images in sitemaps.
Your sitemap will be automatically updated for each separate content you add, without the need to repeatedly create a sitemap.
You will be listed more easily in search engines such as Google, Bing, Yandex, Yahoo, Ask.

### Site Redirects
You can redirect unsecured HTTP traffic to secure HTTPS traffic without touching any code.
When you activate the plugin everything will be configured for you automatically and SSL will be enabled.
The entire site will move to HTTPS using your SSL certificate.
It works with any SSL certificate also you can redirect your website other website

### Seo Image
You can automatically add alt and title tags to the images you add to your website to increase your visibility by search engines.
You can add alt and title tags of images at once.
You can increase the rate of your images appearing in search results.

### Robot Txt Editor
You can determine how your website content will be read by search engine bots from your admin panel.
You can edit the robot.txt file of your website
You can make your edits without the hassle of FTP clients or permissions.

### Error Redirects
You can redirect your pages with 404 errors so that your visitors and search engine robots do not have a bad experience.
You can view the list of broken links on your website without using any SEO tools.
You can make 301,302 and 307 redirects to broken links.

### Error Page Manager
To prevent your visitors from seeing 404 pages as an error, you can have them automatically redirected to a different related content.
You can prevent your users from having a bad experience.
You can automatically redirect pages that give 404 errors to content with relevant and similar URLs.

### Duplicate Manager
You can automatically detect your published content with the same name and same content so that your SEO is not adversely affected.
You can delete duplicate content and redirect to original content.
You can reduce server usage and speed up your website by cleaning up unnecessary content.

### Custom Redirects
You can create and manage redirects for unopened, deleted or broken links on your site to improve SEO and visitor experience.
You can redirect pages with 301,302 and 307 error codes to different pages you want.
You can view the list of links to which you have redirected.

### Visit Limitation
You can prevent unnecessary traffic by limiting the unusual requests of your users on your website.
You can limit your visitors' page visits, searches and comments.
You can exclude any user roles without limitation.
You can exclude query limits for search engines.
You can specify how the rules you specify are stored on your server.
You can direct people who exceed the limits to different addresses.

### Login Url
You can change the standard wp-admin Admin Panel url of your WordPress site.
You can manually change the admin panel login address.
You can block malicious login attempts to your admin panel.

### Google Authenticator
You can secure admin login by adding two factor authentication to your website.
You can protect your website from phishing and password theft.
You can block requests that contain malicious code or content.
By blocking all requests from malicious IPs, you can reduce server load while keeping your site secure.

vCaptcha

You can add Captcha verification to member login, member registration, comment, contact form sections of your website.

You can view and edit all tracking codes from a single page.

When you first set up your WordPress website, it doesn't have many of the features you need for a standard website. To meet these needs, you need to install many paid or free plugins.
